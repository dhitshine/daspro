# Fundamental of Programming Archives

This repository contains the archive of course materials, laboratory exercises, and assignments for the Fundamental of Programming (Dasar Pemrograman) course at Institut Teknologi Sepuluh Nopember (ITS). All programming exercises and solutions in this archive are implemented in the C programming language.

## Repository Structure

The repository is organized into three main directories:

* [Modul](Modul/): Contains code examples, explanations, and exercises for each learning module.
* [Praktikum](Praktikum/): Contains laboratory problem sets, test cases, and solution codes.
* [Tugas](Tugas/): Contains course assignments and homework provided by the lecturer.
